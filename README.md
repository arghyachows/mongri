mongri
=========

### Demo
------
[Mongri on Heroku](http://mongri.herokuapp.com)

### Use Mongri
------
```
$ git clone git@github.com:dongri/mongri.git
$ cd mongri
$ npm install
$ node app.js
```

### Docker

```
$ cd mongri
$ docker-compose up -d
```
